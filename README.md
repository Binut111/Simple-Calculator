# Simple-Calculator
Simple-Calculator repository:

* ✅ index.html - Calculator structure with buttons and display
* ✅ style.css - Beautiful gradient styling with responsive design
* ✅ script.js - Full calculator functionality with keyboard support

## Features included:

* ➕ Basic arithmetic operations (add, subtract, multiply, divide)
* 🔢 Decimal number support
* 🎨 Modern gradient design with smooth animations
* ⌨️ Keyboard support (numbers, operators, Enter to calculate, Backspace to delete, Escape to clear)
* ❌ Clear (C) and Delete (DEL) buttons
* 🛡️ Error handling for invalid expressions
Open the `index.html file` in your browser to start using the calculator!
